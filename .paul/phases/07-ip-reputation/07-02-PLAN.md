  <action>
    In render_selector_view(), AFTER the Raw observations expander block
    (after the closing `st.dataframe(filtered[display_cols]...)` and its `with` block),
    and BEFORE the final closing of the function, insert:

    ```python
    # ── Export Intelligence Report ────────────────────────────────────────────
    st.markdown("---")
    st.markdown("**Export Intelligence Report**")
    _export_date = pd.Timestamp.now().strftime("%Y%m%d")
    _safe_selector = "".join(c if c.isalnum() else "_" for c in sel_value)[:32]

    from modules.report import generate_html_report, generate_csv_export
    from modules.velocity import compute_velocity

    # Gather available enrichment data from session state (populated by enrichment buttons)
    _exp_shodan = st.session_state.get("_last_shodan_df")
    _exp_osm    = st.session_state.get("_last_osm_df")
    _exp_tg     = st.session_state.get("_last_tg_df")
    _exp_vel    = compute_velocity(filtered)

    _col_html, _col_csv = st.columns(2)
    with _col_html:
        _html_report = generate_html_report(
            selector_field=sel_field,
            selector_value=sel_value,
            filtered_df=filtered,
            coloc_df=coloc,
            vel_df=_exp_vel,
            shodan_df=_exp_shodan,
            osm_df=_exp_osm,
            tg_df=_exp_tg,
        )
        st.download_button(
            label="Download HTML Report",
            data=_html_report.encode("utf-8"),
            file_name=f"wraith_intel_{_safe_selector}_{_export_date}.html",
            mime="text/html",
            use_container_width=True,
            key="export_html_btn",
        )
    with _col_csv:
        _csv_data = generate_csv_export(filtered)
        st.download_button(
            label="Download CSV",
            data=_csv_data.encode("utf-8"),
            file_name=f"wraith_obs_{_safe_selector}_{_export_date}.csv",
            mime="text/csv",
            use_container_width=True,
            key="export_csv_btn",
        )
    st.caption("HTML report includes all enrichment data loaded in this session. CSV contains raw observations only.")
    ```

    Also add session state keys for enrichment DataFrames in init_session():
    ```python
    if "_last_shodan_df" not in st.session_state:
        st.session_state._last_shodan_df = None
    if "_last_osm_df" not in st.session_state:
        st.session_state._last_osm_df = None
    if "_last_tg_df" not in st.session_state:
        st.session_state._last_tg_df = None
    ```

    And in render_selector_view(), after each enrichment query succeeds, store the result:
    - After `shodan_df = shodan_search_near(...)` and `not shodan_df.empty`:
      add `st.session_state._last_shodan_df = shodan_df`
    - After `osm_df = overpass_pois(...)` and `not osm_df.empty`:
      add `st.session_state._last_osm_df = osm_df`
    - After `tg_df = search_selector_in_channels(...)` and `not tg_df.empty`:
      add `st.session_state._last_tg_df = tg_df`

    This means the HTML report captures whatever enrichment data the operator
    loaded during the session — if they clicked "Query Shodan", it's in the report.

    Avoid: importing report/velocity at module top-level.
    Avoid: re-computing coloc — already in scope from earlier in the function.
