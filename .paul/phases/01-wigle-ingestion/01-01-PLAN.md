<objective>
## Goal
Create a production-quality WiGLE API client module (`modules/wigle.py`) that authenticates, queries by bounding box, normalizes responses to a WRAITH-compatible DataFrame, handles all known error states, and caches results session-locally to respect free-tier rate limits.

## Purpose
Establishes the data ingestion foundation for WRAITH-WiGLE. All downstream phases (visualization, selector tracking, enrichment) depend on this module producing clean, normalized data. The PoC (`poc_wigle_integration.py`) proved the approach — this plan extracts and hardens it into the production module layer.

## Output
- `modules/wigle.py` — importable WiGLE client
- `modules/__init__.py` — updated to expose wigle module
- `env.example` — WIGLE_USERNAME and WIGLE_TOKEN entries added
</objective>

<task type="auto">
  <name>Task 1: Create modules/wigle.py</name>
  <files>modules/wigle.py</files>
  <action>
    Create a clean production module extracted from poc_wigle_integration.py. Structure:

    **Constants:**
    - WIGLE_SEARCH_URL = "https://api.wigle.net/api/v2/network/search"
    - WIGLE_RESULT_COLUMNS — ordered list of expected response fields

    **wigle_bbox_search(lat, lon, radius_deg, username, token, result_limit=100) -> list[dict]:**
    - HTTP Basic auth via base64(username:token)
    - Add Accept: application/json header
    - Params: latrange1/2, longrange1/2, resultsPerPage (cap at 100)
    - Explicit status code handling BEFORE raise_for_status():
      - 401 → RuntimeError with message about API token (not password)
      - 412 → RuntimeError with message directing to wigle.net/account terms acceptance
      - 429 → RuntimeError with message about rate limit and reducing camera count
    - Returns data.get("results", [])

    **normalize_wigle_response(results: list[dict], near_camera: str = "") -> pd.DataFrame:**
    - Converts raw API results list to DataFrame
    - Ensures all WIGLE_RESULT_COLUMNS exist (fill missing with "")
    - Adds near_camera column
    - Returns empty DataFrame (not raises) if results is empty

    **_session_cache: dict** — module-level dict, keyed by (lat, lon, radius_deg) tuple

    **cached_bbox_search(lat, lon, radius_deg, username, token) -> pd.DataFrame:**
    - Check _session_cache first; return cached value if hit
    - On miss: call wigle_bbox_search, normalize, store in cache, return
    - Cache key: (round(lat,4), round(lon,4), radius_deg)

    **generate_mock_networks(cameras_df: pd.DataFrame, networks_per_camera: int = 8, seed: int = 99) -> pd.DataFrame:**
    - Port directly from poc_wigle_integration.py generate_mock_wigle_networks()
    - Rename to generate_mock_networks for cleaner import
    - No changes to logic

    Docstring at top: module purpose, WiGLE API ref URL, auth note (API token not password).
    Do NOT import streamlit — this is a pure data module.
    Do NOT use st.progress or any UI calls.
  </action>
