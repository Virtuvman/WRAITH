<objective>
## Goal
Create the RAVEN data pipeline: a Shodan/FOFA JSON parser that extracts device records into a unified schema, and a camera classifier that filters/tags records by device type.

## Purpose
Phase 8 establishes the data foundation for all RAVEN views. Downstream plans (raven_media.py, raven_feeds.py, the map view) all read from the unified RAVEN schema defined here. Mock data generator with embedded base64 thumbnails enables full UI development without live Shodan files.

## Output
- `modules/raven_ingest.py` — Shodan/FOFA JSON parser, unified schema, mock generator
- `modules/raven_matcher.py` — camera type classifier and filter
</objective>

  <action>
    Create the RAVEN ingest module. No streamlit imports.

    RAVEN_SCHEMA_COLUMNS = [
        "source", "lat", "lon", "label", "ip", "port", "org",
        "product", "tags", "image_b64", "image_url", "timestamp", "metadata"
    ]

    MOCK_B64_PNG — a minimal valid base64 PNG string (1x1 pixel transparent PNG):
    "iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAYAAAAfFcSJAAAADUlEQVR42mNk+M9QDwADhgGAWjR9awAAAABJRU5ErkJggg=="

    _empty_record() -> dict:
      Return a dict with all RAVEN_SCHEMA_COLUMNS as keys, all values None.
      tags defaults to [], metadata defaults to {}.

    _parse_shodan_record(record: dict) -> dict:
      Extract from Shodan API response format:
        - ip: record.get("ip_str")
        - port: record.get("port")
        - org: record.get("org")
        - product: record.get("product")
        - tags: record.get("tags", [])
        - lat: record.get("location", {}).get("latitude")
        - lon: record.get("location", {}).get("longitude")
        - label: product or org or ip (first non-None)
        - image_b64: record.get("screenshot", {}).get("data") — None if no screenshot
        - image_url: None (Shodan provides embedded b64, not URL)
        - timestamp: record.get("timestamp")
        - source: "shodan"
        - metadata: {"country": record.get("location", {}).get("country_code"), "city": record.get("location", {}).get("city")}
      Return merged with _empty_record() so all columns always present.

    _parse_fofa_record(record: dict) -> dict:
      Extract from FOFA API v2 format (dict per result):
        - ip: record.get("ip")
        - port: record.get("port")
        - org: record.get("as_organization") or record.get("isp")
        - product: record.get("title") or record.get("product")
        - tags: []
        - lat: _safe_float(record.get("latitude"))
        - lon: _safe_float(record.get("longitude"))
        - label: product or ip
        - image_b64: record.get("screenshot") or record.get("base64_screenshot")
        - image_url: None
        - timestamp: record.get("lastupdatetime") or record.get("updated_time")
        - source: "fofa"
        - metadata: {"country": record.get("country_name"), "city": record.get("city")}
      Return merged with _empty_record().

    _safe_float(val) -> float | None:
      Try float(val), return None on exception.

    _detect_format(data: dict) -> str:
      If "matches" in data: return "shodan"
      If "results" in data and isinstance(data["results"][0], dict): return "fofa"
      If "ip_str" in data: return "shodan_single"
      Return "unknown"

    load_raven_file(filepath: str) -> pd.DataFrame:
      Open and json.load the file. Detect format.
      shodan: iterate data["matches"], parse each with _parse_shodan_record
      fofa: iterate data["results"], parse each with _parse_fofa_record
      shodan_single: parse data directly with _parse_shodan_record
      unknown: return empty DataFrame with RAVEN_SCHEMA_COLUMNS
      Catch all exceptions, log to stderr, return empty DataFrame — never raise.

    generate_mock_raven(lat: float = 37.77, lon: float = -122.41, n: int = 10) -> pd.DataFrame:
      Generate n mock camera records around (lat, lon) with ±0.01 degree scatter.
      Use deterministic seed from hash((round(lat,2), round(lon,2), n)).
      Products: cycle through ["Hikvision IP Camera", "Dahua IPC", "Axis M3106",
                               "Vivotek FD9391", "Bosch Flexidome", "traffic camera MJPEG"]
      image_b64: MOCK_B64_PNG for all rows
      source: "shodan" (mock)
      ports: random choice from [80, 443, 8080, 554]
      Return DataFrame with RAVEN_SCHEMA_COLUMNS.

<task type="auto">
  <name>Task 2: Create modules/raven_matcher.py</name>
  <files>modules/raven_matcher.py</files>
  <action>
    Create the camera classifier module. No streamlit imports.

    CAMERA_KEYWORDS = [
        "hikvision", "dahua", "axis", "vivotek", "bosch", "hanwha", "samsung",
        "rtsp", "ipcam", "webcam", "traffic", "cctv", "camera", "surveillance",
        "nvr", "dvr", "milestone", "genetec", "mobotix"
    ]

    CAMERA_PORTS = [554, 8554, 9000, 8080, 37777]  # RTSP and common camera ports

    classify_camera(row: dict | pd.Series) -> bool:
      Check product and label fields (lowercased) against CAMERA_KEYWORDS.
      Also flag if port in CAMERA_PORTS.
      Check tags list for "camera", "iot", "webcam".
      Return True if any match, False otherwise.
      Never raise — catch all exceptions, return False.

    filter_cameras(df: pd.DataFrame) -> pd.DataFrame:
      Apply classify_camera to each row, return filtered DataFrame.
      Return empty DataFrame (same schema) if df is None or empty.

    tag_camera_type(df: pd.DataFrame) -> pd.DataFrame:
      Add "camera_type" column to df (copy, not in-place).
      Values: "traffic" if "traffic" in label/product, "ip_camera" if hikvision/dahua/axis/etc,
              "webcam" if "webcam" in product, "unknown" otherwise.
      Return df with new column. Never raise.
  </action>

  <done>AC-4 satisfied: camera filter identifies camera products and ports correctly</done>
