<objective>
## Goal
Build raven_media.py — the image handling and Folium popup builder for RAVEN. Handles two image sources: base64 strings embedded in Shodan JSON, and local screenshot files from disk. Produces popup HTML and tooltip text consumed by the Phase 10 map view.

## Purpose
Phase 10 (RAVEN Map View) cannot render camera pins without a popup HTML builder. This plan delivers that builder and establishes how images flow from both Shodan JSON (base64) and the user's own screenshot collection (file paths) into the Folium map.

## Output
- `modules/raven_media.py` — image decode, thumbnail, popup HTML, tooltip text, screenshot extractor
- `requirements.txt` — Pillow added as optional dependency
- `scripts/extract_raven_screenshots.py` — CLI tool: JSON file → screenshots to disk
</objective>

    Attempt Pillow import at module level with graceful fallback:
      try:
          from PIL import Image as _PILImage
          _PIL_AVAILABLE = True
      except ImportError:
          _PILImage = None
          _PIL_AVAILABLE = False

    --- CONSTANTS ---
    THUMBNAIL_W = 240
    THUMBNAIL_H = 180
    POPUP_STYLE = inline CSS string for dark-theme popup container:
      background:#0f172a; color:#e2e8f0; padding:8px; border-radius:6px;
      min-width:260px; font-family:monospace; font-size:12px; max-width:280px;

    --- FUNCTIONS ---

    _load_file_bytes(filepath: str) -> bytes | None:
      Open filepath in binary mode, return contents.
      Return None on any exception — never raise.

    _resize_bytes_to_b64(img_bytes: bytes, w=THUMBNAIL_W, h=THUMBNAIL_H) -> str:
      If _PIL_AVAILABLE:
        Open with Image.open(io.BytesIO(img_bytes))
        Call img.thumbnail((w, h), Image.LANCZOS)
        Save to BytesIO as PNG
        Return base64.b64encode(buf.getvalue()).decode("utf-8")
      Else:
        Return base64.b64encode(img_bytes).decode("utf-8")  # no resize, return as-is

    make_thumbnail_b64(row, image_path: str | None = None) -> str | None:
      row can be dict or pd.Series.
      Priority order:
        1. image_path argument (user's own screenshot file from disk)
        2. row["image_b64"] (embedded Shodan base64)
        3. None — return None

      For image_path: load bytes via _load_file_bytes, then _resize_bytes_to_b64
      For image_b64: base64.b64decode(row["image_b64"]) → bytes → _resize_bytes_to_b64
      Return None on any exception — never raise.

    build_popup_html(row, image_path: str | None = None) -> str:
      row can be dict or pd.Series.
      Build self-contained HTML popup string.

      Get thumbnail: thumb_b64 = make_thumbnail_b64(row, image_path)

      Image block:
        If thumb_b64 is not None:
          <img src="data:image/png;base64,{thumb_b64}"
               style="width:240px;max-height:180px;object-fit:cover;
                      border-radius:4px;display:block;margin-bottom:6px;">
        Else:
          <div style="width:240px;height:60px;background:#1e293b;border-radius:4px;
                      display:flex;align-items:center;justify-content:center;
                      margin-bottom:6px;color:#64748b;font-size:11px;">No image</div>

      Metadata block (use html.escape on all values):
        label — bold, color #38bdf8
        IP:port — one line
        Org — one line (truncate to 40 chars)
        Source — one line (shodan / fofa / municipal / youtube)
        Tags — comma-joined, only if non-empty
        Timestamp — smaller font #64748b, only if non-None

      Wrap everything in <div style="{POPUP_STYLE}">...</div>
      Return the HTML string. On any exception return "<div>Error building popup</div>".

    build_tooltip_text(row) -> str:
      row can be dict or pd.Series.
      Return plain text: "{label} | {source} | {lat:.4f},{lon:.4f}"
      Fallback to empty string on exception.

    extract_screenshots(df: pd.DataFrame, output_dir: str = "data/screenshots") -> dict:
      Iterate df rows. For each row where image_b64 is not None/empty:
        filename = "{ip}_{port}.png" where ip has dots replaced with underscores
        filepath = output_dir / filename
        Decode image_b64, write bytes to filepath
        Count successes and skips.
      Create output_dir if it does not exist (pathlib.Path.mkdir parents=True exist_ok=True).
      Return {"extracted": N, "skipped": M, "errors": K, "output_dir": output_dir}
      Never raise — log errors, count them, continue to next row.

    --- requirements.txt ---
    Add after the requests line:
      Pillow>=10.0.0  # optional — enables thumbnail resize in RAVEN media view

    Note: Pillow import failure is handled gracefully — thumbnails still work, just unresized.
