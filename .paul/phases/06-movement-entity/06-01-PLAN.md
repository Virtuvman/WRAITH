<objective>
## Goal
Create modules/velocity.py — a pure data module that calculates movement velocity between WiGLE observations for the same selector (SSID or BSSID), classifies physically impossible movements as anomalies, and produces a per-observation velocity DataFrame ready for display and entity graph wiring.

## Purpose
A BSSID or SSID observed in Toronto, then Lima, then Tokyo within hours is physically impossible for a fixed device — it flags a mobile device (hotspot), spoofed BSSID, or data anomaly. Velocity analysis is the D-ISR tripwire: it surfaces which selectors are moving (and how fast), enabling the operator to distinguish static infrastructure from mobile targets and identify implausible patterns indicating deception or multi-device use.

## Output
- modules/velocity.py — compute_velocity, classify_velocity, VELOCITY_COLUMNS, ANOMALY_THRESHOLD_KMH

    ANOMALY_THRESHOLD_KMH = 500.0  # Faster than commercial aircraft — physically implausible for fixed device

    **_haversine(lat1, lon1, lat2, lon2) -> float:**
    - Returns great-circle distance in km between two lat/lon points
    - Formula: haversine using math.radians, math.sin, math.cos, math.asin, math.sqrt
    - Earth radius: 6371.0 km
    - Pure stdlib math — no external dependencies

    **compute_velocity(df: pd.DataFrame) -> pd.DataFrame:**
    - If df is None or empty: return empty DataFrame with original columns + VELOCITY_COLUMNS
    - Required columns: "trilat", "trilong", "lasttime"
    - If any required column missing: log warning, return df.copy() with VELOCITY_COLUMNS filled with 0/False
    - Sort by lasttime ascending (pd.to_datetime, errors="coerce")
    - Reset index after sort
    - For each row i:
      - If i == 0: speed_kmh=0, distance_km=0, time_delta_h=0, prev_lat=row.trilat, prev_lon=row.trilong
      - Else:
        - distance_km = _haversine(prev.trilat, prev.trilong, curr.trilat, curr.trilong)
        - time_delta_h = (curr_time - prev_time).total_seconds() / 3600.0
        - speed_kmh = distance_km / time_delta_h if time_delta_h > 0 else 0.0
        - prev_lat = prev.trilat, prev_lon = prev.trilong
    - Use vectorized approach where possible — avoid row-by-row Python loop for large DataFrames:
      - shift() to align prev/curr rows
      - apply _haversine rowwise via df.apply(lambda r: _haversine(...), axis=1)
    - Add VELOCITY_COLUMNS to result DataFrame
    - Never raise

    **classify_velocity(df: pd.DataFrame) -> pd.DataFrame:**
    - If df is None or empty or "speed_kmh" not in df: return input unchanged (or empty df)
    - Add is_anomaly (bool) column: speed_kmh > ANOMALY_THRESHOLD_KMH
    - First row (speed_kmh=0) → is_anomaly=False
    - Return df with is_anomaly column added
    - Never raise

    **compute_velocity_by_selector(df: pd.DataFrame, field: str = "netid") -> pd.DataFrame:**

      - Append to results list

    - Return combined DataFrame
    - Never raise
