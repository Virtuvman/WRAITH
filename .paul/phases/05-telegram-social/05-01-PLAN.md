## Goal
Create modules/telegram.py — a pure data module that scrapes public Telegram channel web previews (t.me/s/{channel}) for selector keyword mentions and returns structured results, with no authentication and no new dependencies.

## Purpose
Telegram public channels are a rich OSINT source: security researchers, local observers, and infrastructure watchers frequently post network names, BSSIDs, IP addresses, and location-tagged sightings. Phase 5 surfaces these signals alongside WiGLE observations in the Selector view — connecting wireless observations to open-source human reporting.


## AC-3: Keyword Filtering
```gherkin
Given a channel with mixed message content
When scrape_channel(channel, keyword) is called
Then only messages containing keyword (case-insensitive) are returned
```

```gherkin
Given a keyword and list of channel names
When search_selector_in_channels(keyword, channels) is called
Then returns a combined DataFrame from all channels, deduped by url
And channels that fail or return no results are skipped silently
```

## AC-5: Session Cache
```gherkin
Given scrape_channel called twice with same channel + keyword
When the second call executes
Then the HTTP request is NOT made a second time (cache hit)
```

  <action>
    Create a pure data module (no streamlit imports) using only requests + stdlib.

    **TG_RESULT_COLUMNS** constant:
    `["channel", "date", "text", "url"]`

    **DEFAULT_CHANNELS** constant — list of publicly useful OSINT/security Telegram channels:
    ```python
    DEFAULT_CHANNELS = [
        "cybersecuritynews",
        "osintforall",
        "securityaffairs",
        "netblocks",
        "bellingcat",
    ]
    ```

    **_tg_cache: dict[tuple, pd.DataFrame] = {}** — module-level session cache

    **scrape_channel(channel: str, keyword: str, limit: int = 20, timeout: int = 10) -> pd.DataFrame:**
    - Cache key: (channel.lower().strip(), keyword.lower().strip())
    - On cache hit: return cached result
    - GET https://t.me/s/{channel} with headers:
        User-Agent: "Mozilla/5.0 (compatible; WRAITH-OSINT/1.0; research)"
        Accept: "text/html"
    - On non-200 or exception: log warning, cache and return empty DataFrame
    - Parse HTML using Python stdlib html.parser (HTMLParser subclass):
        - Extract message text from elements matching class "tgme_widget_message_text"
        - Extract date from datetime attribute of elements matching class "tgme_widget_message_date"
        - Extract message URL from href of the date link (format: t.me/{channel}/{msg_id})
      
      Implement a simple HTMLParser subclass that collects:
        - All <a> tags where href matches "https://t.me/{channel}/\d+" — store as current_url
        - All <div> tags where class contains "tgme_widget_message_text" — collect inner text
        - All <time> tags — collect datetime attribute
      
      Simpler alternative if parser complexity is high: use regex on resp.text:
        - Message texts: re.findall(r'class="tgme_widget_message_text[^"]*"[^>]*>(.*?)</div>', html, re.DOTALL)
        - Dates: re.findall(r'<time datetime="([^"]+)"', html)
        - URLs: re.findall(r'href="(https://t\.me/{channel}/\d+)"', html)
        Use the regex approach — simpler and sufficient for this HTML structure.
      
    - Strip HTML tags from extracted text: re.sub(r'<[^>]+>', '', text).strip()
    - Filter rows where text contains keyword (case-insensitive, partial match)
    - Build DataFrame with TG_RESULT_COLUMNS; limit to `limit` most recent rows
    - Store result in cache before returning
    - Never raise

    **search_selector_in_channels(keyword: str, channels: list[str] | None = None, limit_per_channel: int = 10) -> pd.DataFrame:**
    - If channels is None: use DEFAULT_CHANNELS
    - Call scrape_channel for each channel with limit=limit_per_channel
    - Concatenate non-empty results
    - Deduplicate by "url" column
    - Return combined DataFrame sorted by date desc (if date column parseable)
    - If all channels return empty: return empty DataFrame with TG_RESULT_COLUMNS
    - Never raise

    **clear_tg_cache() -> None:** clears _tg_cache dict

    Avoid: importing streamlit, BeautifulSoup, lxml, or any non-stdlib/non-requests library.
    Use: re module (stdlib), html module for unescape (stdlib), requests.
    Note: t.me/s/ only shows ~20 most recent posts per channel — this is expected behavior.
