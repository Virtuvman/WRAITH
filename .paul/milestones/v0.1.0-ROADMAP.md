# Roadmap: WRAITH-WiGLE

## Overview

WRAITH-WiGLE is built in focused phases: first establish the WiGLE ingestion and visualization core, then layer in cross-source enrichment (Shodan, OSM, Telegram), then build the behavioral analytics layer (movement velocity, entity graph), and finally deliver the full selector-tracking ISR dashboard.
